# cybersecurity_awareness_chatbot
The Cipher Cybersecurity Awareness Chatbot is a C# console application designed to educate users on digital safety.
The program utilizes a modular object-oriented structure to manage multimedia assets and security logic.
An automated voice greeting welcomes the user via the voice_greeting class and a PCM WAV file.
The ascii_logo class dynamically converts a PNG image into a grayscale ASCII graphic within the terminal.
User engagement is driven by a simulated typing effect that displays text word-by-word for a conversational feel.
The welcome_and_username class manages the user interface, including custom text colors and name collection.
Core functionality resides in the prompt_and_search class, which uses keyword-matching to answer security questions.
The bot provides expert advice on topics such as password complexity, VPN usage, and multi-factor authentication.
For complex threats like phishing, the system pulls from a randomized array of tips to ensure varied educational content.
This application effectively combines visual art, audio feedback, and structured data to make cybersecurity learning accessible.
